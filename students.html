MARCH 14TH BEFORE CLASS. You MUST have your updated portfolio done and functional before then as you will be grading one anothers. Please note your ninjas are here to help you get everything working before then.

Final Project:

Your final project can be anything really, so long as it has sufficient enough challenge and builds on what you have been learning in this class. Topics can include:

  • Redesigning your portfolio from scratch into the epic-tome of epic-ness, using Javascript libraries.
  • Making a landing page for a fake startup. Can include things like parallax scrolling, media queries, etc.
  • Combine this with a final project for Software Design
  • Make a mini, but super classy app with something like AngularJS (please feel free to ask for help)
Projects will ideally be done in teams, however it is fine if you want to do it individually.
